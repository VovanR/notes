# RegExp

----

## Вперед смотрящий
Выберет `foo`, после которого идет `bar`
```
foo(?= bar)
```
не идет `bar`
```
foo(?! bar)
```



## Флаги
```
\...\gim
```
- `g` - Глобально (без него выбирается первое вхождение)
- `i` - Регистронезависимый
- `m` - Многострочный



## Не жадный поиск
```
a{2,10}?e
```
`?` - Включает нежадность



## Запоминающие группы (в скобках)
Незапоминающие - `?:`
```
java(?:script)?
```
